# board
MVC 패턴 게시판
 - 국비 교육간 개인프로젝트(공통)
 - MVC패턴 웹페이지 개발
 - 기능 : 게시판, 로그인, 회원가입, 이미지 게시판(파일업로드), 메일보내기
 - 언어 : JAVA, JTSL, EL, HTML/CSS, JavaScript, J-Query, BootStrap
 - 개발도구 : Tomcat 8.0, Eclipse, Filezilla
